# Finding File Template — Deprecated

Per-finding markdown files are no longer the primary review artifact in Khuym reviewing.

Review issues now become beads directly, and the full review detail lives in the bead body.

Use:

- `review-bead-template.md` — active template and creation contract for review beads
- `.khuym/findings/learnings-candidates.md` — still allowed as the single session-level summary artifact written by the learnings-synthesizer

The old `{id}-pending-{priority}-{slug}.md` file contract is retired and should not be used for new reviewing work.
